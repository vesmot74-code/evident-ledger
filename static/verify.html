Deterministic Event Ledger

Верификатор
цепочки событий

Проверка криптографической целостности цепочки по chain_id — merkle root, подпись, количество событий.

Введите chain_id

UUID цепочки, полученный при её создании (evident new-chain или GUI).

Введите SHA-256 хэш файла

Хэш вычисляется локально (sha256sum / evident hash), сам файл никогда не отправляется на сервер.
